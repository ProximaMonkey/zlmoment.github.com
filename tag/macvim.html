安装MacVim和Python

下列两种方式中的任一种均可用来安装MacVim。

方式一：通过Mac ports

首先确保你的Mac ports是最新版：

1 $ sudo port -v selfupdate

安装：

$ sudo port install macvim +cscope +python26

方式二：自己编译

$ git clone git://github.com/b4winckler/macvim.git
$ cd macvim/src
$ ./configure --enable-pythoninterp --with-macsdk=10.6
