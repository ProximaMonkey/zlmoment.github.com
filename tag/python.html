Python中的装饰器Decorator

Stackoverflow上面有一个问题（Here），问题是这样的：

如果使用如下的代码：

@makebold
@makeitalic
def say():
    return "Hello"

打印出如下的输出：

<b><i>Hello</i></b>

你会怎么做？最后给出的答案是：

def makebold(fn):
    def wrapped():
        return "<b>" + fn() + "</b>"
    return wrapped

def makeitalic(fn):
    def wrapped():
        return "<i>" + fn() + "</i>"
    return wrapped

@makebold
@makeitalic
def hello():
    return "hello world"

组件介绍

Scrapy Engine（Scrapy引擎）

Scrapy引擎主要负责控制各个组件之间的数据处理过程，并在特定动作发生时触发相应的事件。

Scheduler（调度控制器）
