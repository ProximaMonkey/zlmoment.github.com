Python中的装饰器Decorator

Fri 11 May 2012 by Zhaoyu Li

Stackoverflow上面有一个问题（Here），问题是这样的：

如果使用如下的代码：

@makebold
@makeitalic
def say():
    return "Hello"

打印出如下的输出：

<b><i>Hello</i></b>

你会怎么做？最后给出的答案是：

def makebold(fn):
    def wrapped():
        return "<b>" + fn() + "</b>"
    return wrapped

def makeitalic(fn):
    def wrapped():
        return "<i>" + fn() + "</i>"
    return wrapped

@makebold
@makeitalic

    return "hello world"

组件介绍

Scrapy Engine（Scrapy引擎）
