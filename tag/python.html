SUDS错误和实例

 Tue 13 November 2012 by Zhaoyu Li

没想到最近项目方给了一个WebService接口，硬着头皮拿Pyhton来解析，终于迈过了第一大步，后面的过程就会轻松很多了。

错误信息：

Traceback (most recent call last):
  File "test.py", line 12, in <module>
    client = Client(url, doctor=d)
  File "/Library/Python/2.7/site-packages/suds-0.4-py2.7.egg/suds/client.py", line 112, in __init__

  File "/Library/Python/2.7/site-packages/suds-0.4-py2.7.egg/suds/wsdl.py", line 220, in build_schema
    self.schema = container.load(self.options)
  File "/Library/Python/2.7/site-packages/suds-0.4-py2.7.egg/suds/xsd/schema.py", line 95, in load
    child.dereference()
  File "/Library/Python/2.7/site-packages/suds-0.4-py2.7.egg/suds/xsd/schema.py", line 323, in dereference
    midx, deps = x.dependencies()
  File "/Library/Python/2.7/site-packages/suds-0.4-py2.7.egg/suds/xsd/sxbasic.py", line 422, in dependencies

suds.TypeNotFound: Type not found: '(schema, http://www.w3.org/2001/XMLSchema, )

解决办法：

from suds.client import Client
from suds.xsd.doctor import ImportDoctor, Import

url = 'https://solaranywhere.com/service.asmx?WSDL'
imp = Import('http://www.w3.org/2001/XMLSchema', location='http://www.w3.org/2001/XMLSchema.xsd')
imp.filter.add('http://www.solaranywhere.com/')
d = ImportDoctor(imp)
client = Client(url, doctor=d)
print client

第六行折腾了半天有木有。

@makebold

组件介绍

Scrapy Engine（Scrapy引擎）

Scrapy引擎主要负责控制各个组件之间的数据处理过程，并在特定动作发生时触发相应的事件。

Scheduler（调度控制器）

调度控制器从Scrapy引擎接受请求，并将请求列入队列，当Scrapy引擎发出请求时再返回给引擎。
