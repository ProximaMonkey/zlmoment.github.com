Python中的装饰器Decorator

Stackoverflow上面有一个问题（Here），问题是这样的：

如果使用如下的代码：

@makebold
@makeitalic
def say():
    return "Hello"

打印出如下的输出：

<b><i>Hello</i></b>

你会怎么做？最后给出的答案是：

def makebold(fn):
    def wrapped():
        return "<b>" + fn() + "</b>"
    return wrapped

def makeitalic(fn):
    def wrapped():
        return "<i>" + fn() + "</i>"
    return wrapped

@makebold
@makeitalic
def hello():
    return "hello world"

print hello() ## 返回 ...
