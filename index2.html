时间正在将过往一丝一丝地抹去，我想在淡忘之前，再次品味一下那年的生活。

如果使用如下的代码：

@makebold
@makeitalic
def say():
    return "Hello"

打印出如下的输出：

<b><i>Hello</i></b>

你会怎么做？最后给出的答案是：

def makebold(fn):
    def wrapped():
        return "<b>" + fn() + "</b>"
    return wrapped

def makeitalic(fn):
    def wrapped():
        return "<i>" + fn() + "</i>"
    return wrapped
